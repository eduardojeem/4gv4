# Changelog

Todos los cambios notables de este proyecto serán documentados en este archivo.

El formato está basado en [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
y este proyecto adhiere a [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Sistema completo de gestión empresarial
- Dashboard administrativo con métricas en tiempo real
- Módulo de gestión de inventarios
- Sistema POS integrado
- Módulo de reparaciones técnicas
- Gestión de clientes con CRM
- Reportes y analytics interactivos
- Modo oscuro completo
- Diseño responsive
- Testing completo con Vitest y Playwright
- Storybook para documentación de componentes

### Changed
- Migración a Next.js 15 y React 19
- Actualización a TypeScript 5.9
- Implementación de Turbopack para desarrollo

### Fixed
- Optimizaciones de performance en componentes
- Mejoras de accesibilidad
- Corrección de errores de renderizado
- Optimización de bundle size

## [0.1.0] - 2024-12-29

### Added
- Configuración inicial del proyecto
- Estructura base de componentes
- Integración con Supabase
- Configuración de CI/CD
